  1. $all

    在查询匹配数组中含有的所有元素

  2. $size

    选出文档中的数组含有指定多少个元素的文档

    value = 1,则表示数组中含有元素的个数为1，也就是数组的大小

  3. $elemMatch

    如果数组字段中的元素同时符合$elemMatch中的条件，则选出该文档
