JS入门 - what is JavaScript

leollon
2017-08-28

HTML 和 CSS 是标记语言。标记语言用于描述和定义文档中的元素。JavaScript 是编程语言。
编程语言用于向机器发出指令。编程语言可用于控制机器的行为和表达算法。
