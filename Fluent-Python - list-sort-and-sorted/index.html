Fluent-Python - list.sort() and sorted()

leollon
2017-09-06
Python, notes

: 代表参数

list.sort() 和 sorted()异同：

异：

  • list.sort()

    不创建新序列，返回值为None

    不接收可迭代的对象

  • sorted()

    返回值一个排序后的新创建的序列。

    不可接收可迭代对象

同：
函数参数可以有：

  1. reverse

    reverse=True: 递减排序
    reverse=False: 递增排序

  2. key

    key=function_name: 根据这个函数的提供的信息进行排序
